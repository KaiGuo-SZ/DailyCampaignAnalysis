# 获客总表分析工具｜产品版

## 使用方式
- 打开方式：直接双击 `获客总表分析工具_产品版/index.html`
- 导入数据：右上角点击“导入 CSV”，选择清洗后的数据文件（示例：`数据/英语项目获客总表_清洗后.csv`）

## CSV 要求（必需列）
- 项目
- 营期（或 期）
- 投放平台
- 投放总成本
- 获客量
- 添加数
- L1总单量
- L1总产值

## 可选列（缺失按 0 处理）
- DAY3、DAY4、DAY5、追单（用于总览页的 D3/D4/D5/D6-7 转率）
- 标准/非标 或 渠道类型（用于区分“标准渠道/非标渠道”；缺失时默认按“非标渠道”处理）
- 运营年/运营月（如果 CSV 里有，会自动启用年度/月筛选；没有则隐藏）

## 页面说明
- 总览：按渠道类型汇总 + 投放平台明细（支持标准/非标切换）
- 渠道结构：按项目对比标准/非标占比与关键指标；支持下钻到投放平台明细与跨项目对比
- 标准渠道 / 非标渠道：渠道总览 + 投放平台总览 + 平台明细图表
- 计算器：
  - 分析营期：项目+期 单选
  - 对照营期：项目+期 多选
  - 按投放平台取并集对齐，按对照获客占比降序排序
  - 差异公式：
    - 获客占比差异 = 分析获客占比 - 对照获客占比
    - 转率差异 = 分析转率 - 对照转率
    - 转化人数差异 = 分析(该平台添加数 × 分析转率) - 分析(总添加数 × 对照获客占比 × 对照转率)
    - 产值差异 = 转化人数差异 × 对照平均客单价
  - 平均客单价 = L1总产值 / L1总单量（除数为 0 返回 0）
  - 导出 CSV：UTF-8（带 BOM），Excel 可直接打开
